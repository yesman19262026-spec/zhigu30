# 知股30天：AKShare 行情数据接入方案

> 状态：已确认可使用 AKShare 作为首版市场数据来源。  
> 原则：务实使用、清楚标注、限制频率、允许替换、异常可降级。

## 1. 为什么需要独立数据服务

AKShare 是 Python 库，微信小程序端不能直接运行。整体结构为：

`微信小程序 → HTTPS 行情服务 → 缓存 → AKShare → 底层公开数据源`

小程序只接收整理后的市场观察数据，不直接持有 AKShare 逻辑，也不直接请求底层财经网站。

## 2. 首版接口

| 接口 | 内容 | 建议刷新 |
|---|---|---|
| `/api/market/status` | 当前交易状态、交易日、服务状态 | 60 秒 |
| `/api/market/indices` | 主要指数价格、涨跌、成交额、时间戳 | 30–60 秒 |
| `/api/market/breadth` | 上涨/下跌/平盘、涨跌停家数 | 60 秒 |
| `/api/market/sectors` | 行业涨跌分布和领涨/领跌行业 | 2–5 分钟 |
| `/api/market/turnover` | 沪深京成交额及历史均值比较 | 1–2 分钟 |
| `/api/market/summary` | 面向投教的结构化事实摘要 | 5 分钟/收盘后 |

接口返回统一包含：

- `data_time`：底层数据时间。
- `fetched_at`：服务获取时间。
- `source`：AKShare。
- `upstream_source`：底层公开来源。
- `freshness`：实时、准实时、延时、收盘或缓存。
- `is_stale`：数据是否超过预期时间。
- `disclaimer`：行情仅用于市场观察和投资者教育。

## 3. 缓存与降级

- 同一份全市场数据只由服务端定时获取一次，所有用户共享缓存，避免每位用户触发底层请求。
- 盘中指数缓存 30–60 秒，市场宽度缓存 60 秒，行业数据缓存 2–5 分钟。
- AKShare 或底层来源异常时，返回最近一次成功快照，并明显显示“数据延迟”。
- 快照超过可接受时限时停止展示涨跌颜色，改为“数据暂不可用”，避免旧数据被误认为实时。
- 内置一份虚构教学样本，用于开发、演示和服务完全不可用时的学习功能。

## 4. 数据使用边界

- 不直接逆向或请求财经网站隐藏接口；统一通过 AKShare 已公开的库接口调用。
- 不向用户提供全市场原始行情文件、批量下载或二次 API 转售。
- 页面展示 AKShare、底层来源、时间戳和延迟状态。
- 只提供市场观察、聚合统计和投教解释，不提供个股实时买卖信号。
- 定期检查 AKShare 接口文档、使用说明及底层来源变化；出现明确禁止公开展示的条款时停用对应接口。
- 保留数据适配层，任何单一接口失效都不影响课程、题库和学习记录。

## 5. 建议的数据字段

### 指数

`code, name, last, change, change_pct, open, high, low, prev_close, volume, amount, data_time`

### 市场宽度

`up_count, down_count, flat_count, limit_up_count, limit_down_count, total_count, data_time`

### 行业表现

`sector_code, sector_name, change_pct, up_count, down_count, amount, rank, data_time`

### 成交活跃度

`market, amount, prev_amount, avg_5d, avg_20d, ratio_5d, ratio_20d, data_time`

## 6. 前端展示规则

- 页面标题使用“市场观察”，不使用“交易终端”。
- 顶部始终显示“数据截至 HH:mm:ss”。
- 只有数据在预期时效内才使用“准实时”；不能仅因为页面刚刷新就标记实时。
- 聚合数据与教学解释分区显示，教学解释使用“可能、通常、需结合”等条件化表达。
- 不设涨幅榜个股一键跟买、盘口跟单、主力资金确定性信号或收益预测。

## 7. 开发与上线

### 开发阶段

- 本机运行 Python 行情服务，成本为零。
- 小程序开发环境连接本地或测试 HTTPS 地址。
- 用固定快照为页面和自动化测试提供稳定数据。

### 上线阶段

- 需要可被微信小程序访问的 HTTPS 域名和稳定 Python 运行环境。
- 优先选择有免费额度且允许常驻服务的托管方式；免费额度结束前评估真实访问量。
- 设置请求频率、缓存命中率、异常率和每日流量告警，避免意外费用。
- 若零成本托管无法维持，市场页降级为定时快照，不影响全部免费课程继续使用。

## 8. 可替换适配器

内部统一定义 `MarketDataProvider`：

- `AkshareProvider`：首版默认。
- `SnapshotProvider`：固定或定时快照降级。
- `LicensedProvider`：未来的持牌数据商或公益合作接口。

前端不感知具体供应商，切换数据来源时保持接口字段不变。
