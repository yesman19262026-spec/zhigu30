# 知股30天：课程内容来源与审核规范

> 目标：允许广泛吸收有效、可靠的知识，同时保证每一条课程结论可追溯、可复核、可更新、可纠错。

## 1. 来源准入原则

一项资料可以进入课程候选库，至少应满足以下条件：

1. 能找到原始发布页面、正式出版信息或论文全文信息。
2. 能确认作者、发布机构或责任主体。
3. 能确认发布日期、版本和适用范围。
4. 内容与课程目标直接相关，不以荐股、营销或收益承诺为目的。
5. 使用方式符合版权和数据授权要求。

“有认证”不是唯一标准。监管规则、法定公告、会计准则、专业组织知识体系、正式教材和同行评审研究的认证方式不同，应按证据类型分别评估。

## 2. 来源等级

| 等级 | 类型 | 主要用途 | 使用要求 |
|---|---|---|---|
| S | 法律法规、监管/交易所规则、法定披露、政府原始数据、会计准则 | 规则和事实 | 优先引用最新版原文 |
| A | 权威投教、专业组织标准、正式教材 | 概念与方法 | 涉及中国市场规则时由 S 级校准 |
| B | 同行评审论文、大学和研究机构公开研究 | 研究发现与局限 | 说明样本、时期、方法和适用边界 |
| C | 财经媒体、券商公开资料、行业报告、自媒体 | 案例线索 | 不得作为关键结论的唯一依据 |
| 禁用 | 匿名荐股、无原始出处、稳赚承诺、伪造回测、盗版内容 | 不使用 | 直接排除并记录原因 |

## 3. 单节课程审核清单

- [ ] 学习目标明确，并能在 15 分钟左右完成。
- [ ] 术语定义与原始来源一致。
- [ ] 涉及 A 股规则时引用现行中国规则。
- [ ] 涉及财报时引用现行会计准则或法定披露。
- [ ] 涉及宏观数据时引用政府或央行原始数据。
- [ ] 涉及技术分析或统计规律时说明失败条件、样本偏差和交易成本。
- [ ] 事实、观点、研究发现和虚构案例已明确区分。
- [ ] 关键结论至少完成一次独立交叉核验。
- [ ] 无个股买卖建议、目标价、收益承诺或变相荐股。
- [ ] 无未经授权复制的正文、图片、表格、课程或题库。
- [ ] 已标注内容版本、审核人和下次复核日期。

## 4. 来源台账字段

每条来源至少记录：

| 字段 | 示例 |
|---|---|
| source_id | SRC-0001 |
| 来源等级 | S |
| 机构/作者 | 上海证券交易所 |
| 标题 | 上海证券交易所交易规则（2026年修订） |
| 原文链接/出版信息 | 原始 URL 或 ISBN/DOI |
| 发布日期 | 2026-04-24 |
| 引用日期 | 2026-07-10 |
| 适用市场 | 上交所股票市场 |
| 适用主题 | 交易时间、申报、行情信息 |
| 课程用法 | 转述规则并设计虚构案例 |
| 版权/授权 | 仅必要引用并链接原文 |
| 交叉验证 | 已与交易所配套通知核验 |
| 失效触发条件 | 新版规则发布或条文实施状态改变 |
| 下次复核日期 | 2026-10-10 |
| 审核状态 | 已通过/待复核/停用 |
| 审核人 | 姓名或角色 |
| 备注 | 暂缓实施条文需单独标注 |

## 5. 推荐复核周期

- 交易规则、监管政策、行情授权：每月检查一次，并订阅发布机构更新。
- 会计准则、上市规则、适当性制度：每季度检查一次。
- 宏观数据：按原发布频率更新，并保留历史版本。
- 教材中的稳定概念：每年复核一次。
- 学术研究：不因新论文自动推翻旧内容，按证据积累情况复核。
- 任何被用户指出可能错误的内容：立即标记待审，在核验完成前显示争议提示或暂时下线。

## 7. 内容纠错流程

`用户或编辑提交问题 → 标记受影响课程 → 回到原始来源核验 → 第二人复核 → 修订并记录差异 → 发布更新说明 → 必要时通知已学用户复习`

任何内容都不能因为“专家说过”而免于复核。课程的可信度来自证据链、版本管理和公开纠错，而不是个人权威。
